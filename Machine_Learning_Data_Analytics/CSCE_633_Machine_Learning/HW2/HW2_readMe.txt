Homework 2: Decision Tree Implementation and Support Vector Machine Library Performance Comparison
Project Description
The homework assignment demonstrates:
How to implement a decision tree to classify benign versus malignant tumors based on the Breast Cancer Wisconsin data set using a set of identified features.
How to implement a support vector machine, starting from data pre-processing of training and test data , and comparative evaluation of  SVM types (linear versus kernel). For more information, see hw2.pdf included in the directory.
