Homework 1: Linear Regression and Classification Implementation
Project Description
The homework assignment demonstrates how to calculate the minimum residual sum of squares of a 1 dimensional linear regression model, how to compute the optimal learning step size in a gradient descent to a target function, and implementing a KNN (k-nearest neighbor) classifier for a prediction task given a data set. For more information, see hw1.pdf included in the HW1 directory.
